## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | File name |
| `description` | `string` | - | File description or size |
| `icon` | `React.ReactNode` | `<FileCode />` | Custom file icon |
| `className` | `string` | - | Additional CSS classes |
