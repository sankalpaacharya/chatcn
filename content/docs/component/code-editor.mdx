## Custom Languages

Pass your own languages array via the `languages` prop:

```tsx
const MY_LANGUAGES: Language[] = [
  { value: "python", label: "Python", icon: "https://..." },
  { value: "rust", label: "Rust" }, // icon is optional
]

<CodeEditor languages={MY_LANGUAGES}>
```
