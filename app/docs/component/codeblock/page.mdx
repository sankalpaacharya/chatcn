## 🎨 Line Number
if you want to display line number, add this to your `global.css` file

<CodeBlock lang="css">{`code {
  counter-reset: step;
  counter-increment: step 0;
  margin: 0;
  width: 100%;
}

code .line::before {
  content: counter(step);
  counter-increment: step;
  width: 1rem;
  margin-right: 1.5rem;
  display: inline-block;
  text-align: right;
  color: gray
}`}</CodeBlock>
