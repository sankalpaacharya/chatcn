## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Markdown content to render |
| `className` | `string` | - | Additional CSS classes |
| `theme` | `string` | - | Code block theme (`"dark"` or `"light"`) |
