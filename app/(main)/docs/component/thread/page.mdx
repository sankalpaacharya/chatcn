## Props

### Thread

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Thread content and actions |
| `className` | `string` | - | Additional CSS classes |

### ThreadContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Thread text content |
| `className` | `string` | - | Additional CSS classes |

### ThreadAction

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Action item content |
| `className` | `string` | - | Additional CSS classes |

### ThreadActions

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Action items (ThreadAction components) |
| `className` | `string` | - | Additional CSS classes |
