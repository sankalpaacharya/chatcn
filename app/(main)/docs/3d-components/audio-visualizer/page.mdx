# Audio Visualizer

The **AudioVisualizerExperience** component is an immersive, real-time 3D audio visualization tool built with **React Three Fiber** and **Three.js**. It transforms live microphone input into stunning 3D visualizations, offering users a dynamic and interactive audio experience.
