## Usage

The `ApplicationManager` component doesn't require any props. It automatically handles keyboard shortcuts and state internally.

```tsx
import { ApplicationManager } from "@/components/chatcn/system/applications-manager"

export default function App() {
  return <ApplicationManager />
}
```

Press `Ctrl+K` (or `⌘+K` on Mac) to open the application launcher.
