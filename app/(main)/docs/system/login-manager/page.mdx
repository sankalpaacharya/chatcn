## Props

### LoginManager

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `wallpaper` | `string` | `"/images/jump.png"` | Background image URL for the login screen |
| `portal` | `boolean` | `false` | If true, renders the login screen in a portal |
